# Lista de Tareas con Tkinter

Aplicación desarrollada en Python utilizando Tkinter que permite gestionar tareas diarias.

## Funcionalidades

- Agregar tareas
- Marcar tareas como completadas
- Eliminar tareas

## Atajos de teclado

- Enter: Agregar tarea
- C: Completar tarea
- D o Delete: Eliminar tarea
- Escape: Salir de la aplicación

## Estructura del proyecto

- modelos: contiene la clase Tarea
- servicios: lógica del sistema
- ui: interfaz gráfica
- main.py: punto de inicio
